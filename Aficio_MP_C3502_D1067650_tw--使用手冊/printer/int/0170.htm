規格

解析度

600 dpi 600 dpi / 1200 dpi 1200 dpi

列印速度

機型1：

  • 彩色：20ppm

  • 黑白：20ppm

  • 彩色：25ppm

  • 黑白：25ppm

列印速度因機器而異。請查看您的機器類型。請參閱關於本機。

介面

標準：

  • 乙太網路介面（100BASE－TX／10BASE－T）

  • USB 2.0

選購件：

  • 超高速乙太網路介面（1000BASE－T／100BASE－TX／10BASE－T）

  • IEEE 1284平行介面

網路協定

TCP／IP（IPv4、IPv6）、IPX／SPX

印表機語言

標準：PCL 5c／6

選購件：PostScript 3、PDF、PictBridge

字型

PCL 5c／6：

  • 45種字型、13種國際字型。

PostScript 3：

  • 136種字型

記憶體

1.5 GB

硬碟

160 GB

USB介面（標準）

支援的作業系統：

  • Windows 2000／XP／Vista／7、Windows Server 2003／2003 R2／2008／2008 R2、Mac OS 10.3.3或更新版本

傳送規格：

  • USB 2.0標準

可連接的裝置：

  • 符合USB 2.0標準的裝置

  • 解析度為1200 dpi時可能會降低列印速度。

  • 使用Mac OS X作業系統時，只能使用標準的USB介面。不支援選購的USB介面迴路板。

  • 在Mac OS X 10.3.3或更新版本中使用USB介面（標準）時，可支援USB 2.0。
