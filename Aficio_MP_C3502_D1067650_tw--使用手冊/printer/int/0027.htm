Windows Vista、Windows Server 2008－WSD連接埠

請結束目前執行的所有應用程式。

在[開始]選單上，按一下[網路]。

出現[網路]視窗，自動開始搜尋裝置。

在印表機圖示上按一下滑鼠右鍵，然後再按[安裝]。

如果出現[使用者帳號控制]對話方塊，請按[繼續]。

出現[發現新硬體]對話方塊。

按一下[找到並安裝驅動程式軟體（建議選項）]。

如果出現[使用者帳號控制]對話方塊，請按[繼續]。

按一下[瀏覽我的電腦搜尋驅動程式軟體（進階）]。

將隨附的光碟片置入電腦的光碟機。

如果出現[自動播放]對話方塊，請按一下[關閉]按鈕。

按一下[瀏覽...]，指定印表機驅動程式的位置。

如果光碟機代號為D槽，印表機驅動程式原始檔案儲存在下列位置：

  • PCL 5c

    D:\DRIVERS\PCL5C\XP_VISTA\（語言）\DISK1

  • PCL 6

    D:\DRIVERS\PCL6\XP_VISTA\MUI\DISK1

PCL 6印表機驅動程式會以作業系統指定的語言安裝。

按一下[下一步]。

如果出現[Windows無法驗證此驅動程式軟體的發行者]訊息，請按[仍安裝此驅動程式軟體]。

按一下[關閉]。

如果安裝成功，與WSD連接埠相連印表機的圖示，即會新增到[印表機]視窗。

  • 「WSD」後面的連接埠名稱採用隨機字元字串。無法任意更改。

  • 若要停止安裝選取的驅動程式，請在完成安裝之前按一下[取消]。當重新安裝WSD連接埠時，在[網路]視窗中以滑鼠右鍵按一下本機圖示，然後按一下[解除安裝]。
