用作NetWare列印伺服器／遠端印表機

本節說明如何設定用作NetWare用戶端的Windows電腦。

  • 安裝驅動程式需要使用管理印表機權限。請以系統管理員群組成員的身份登入。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

  • 在Windows Vista／7和Windows Server 2008／2008 R2環境中無法使用IPX／SPX（NetWare）。

  • 在IPv6環境中，無法使用NetWare伺服器。

下列說明是假設NetWare用戶端已安裝在客戶電腦，且正確設定了NetWare伺服器環境。請先安裝所有必要的用戶端應用程式，再操作此步驟。

請登入NetWare檔案伺服器，以設定印表機驅動程式。

在範例中使用下列參考值：

  • NetWare版本：4.1

  • 檔案伺服器名稱：CAREE

  • 佇列名稱：R-QUEUE

請結束目前執行的所有應用程式。

將隨附的光碟片置入電腦的光碟機。

安裝程式隨即啟動。

選擇介面語言，然後按下[確定]。

按一下[PCL 印表機驅動程式]。

[授權同意書]對話方塊中出現軟體的授權同意書。

詳閱同意書後，按一下[我接受此同意書。]，再按[下一步>]。

從[選擇程式]對話方塊中選擇要使用的印表機驅動程式。

可以選擇數個印表機驅動程式。

PCL 6印表機驅動程式會以作業系統指定的語言安裝。

請按[下一步>]。

選擇[印表機名稱]核取方塊，以選擇所要使用的機器型號。

在[更改「印表機名稱」的設定]對話方塊中，可以變更機器的名稱。

按兩下機器的名稱以顯示印表機設定。

視作業系統、機器的型號和連接埠而定，在[註解]、[驅動程式]、[連接埠]中顯示的細節將有所不同。

按一下[連接埠]，再按[新增]。

按一下[網路印表機]，再按[確定]。

在網路樹狀目錄中，按兩下NetWare檔案伺服器名稱。

建立的佇列隨即顯示。

選擇列印佇列，再按[確定]。

檢查選擇的印表機連接埠是否顯示在[連接埠]中。

按一下[繼續]。

印表機驅動程式開始安裝。

在[選擇程式]對話方塊中，按一下[完成]。

安裝完成後，選擇重新啟動電腦或稍後啟動，再按[完成]。

重新啟動電腦以便完成安裝。

重新啟動電腦後，從[印表機和傳真]視窗開啟印表機內容。

在Windows 2000或Windows Server 2003／2003 R2中，開啟[印表機]視窗及印表機內容。

按一下[NetWare設定]索引標籤。

在Windows Server 2003／2003 R2中，從[印表機設定]索引標籤中，取消[換頁]和[啟用橫幅]核取方塊。

請勿選擇這些核取方塊，因為他們由印表機驅動程式自動選擇。如果您選擇這些核取方塊，則印表機無法正確進行列印。

按[確定]關閉印表機內容對話方塊。

  • 某些作業系統設定可能無法使用自動執行。此時，啟動光碟片根目錄中的「Setup.exe」。

  • 若要停止安裝所選取的驅動程式，請在安裝完成之前，按一下[取消]。

  • 在預設狀態下，協定被設定為停用。使用Web Image Monitor或telnet，在控制面板上啟用通訊協定。
