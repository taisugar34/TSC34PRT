變更SmartDeviceMonitor for Client的連接埠設定

依下列步驟改變SmartDeviceMonitor for Client的設定，例如TCP／IP逾時、互助／平行列印及印表機群組。

Windows 2000：

在[開始]選單中指向[設定]，然後再按[印表機]。

出現[印表機]視窗。

按一下所要使用的印表機的圖示。在[檔案]選單上，按一下[內容]。

在[連接埠]索引標籤中，按一下[設定連接埠...]。

出現[設定]對話方塊。

Windows XP、Windows Server 2003／2003 R2：

在[開始]選單上，按一下[印表機和傳真]。

出現[印表機和傳真]視窗。

按一下所要使用的印表機的圖示。在[檔案]選單上，按一下[內容]。

按一下[連接埠]索引標籤，然後按[設定連接埠...]。

出現[連接埠設定]對話方塊。

Windows Vista、Windows Server 2008：

在[開始]選單上，按一下[控制台]。

出現[控制台]視窗。

按一下「硬體和音效」中的[印表機]。

出現[印表機]視窗。

在所要使用的印表機圖示上按一下滑鼠右鍵，再按[內容]。

按一下[連接埠]索引標籤，然後按[設定連接埠...]。

出現[連接埠設定]對話方塊。

Windows 7、Windows Server 2008 R2：

在[開始]選單上，按一下[裝置和印表機]。

隨即顯示[裝置和印表機]視窗。

在所要使用的印表機圖示上按一下滑鼠右鍵，再按[印表機內容]。

出現印表機內容。

按一下[連接埠]索引標籤，然後按[設定連接埠...]。

出現[連接埠設定]對話方塊。

  • 針對IPP，可以設定使用者、代理伺服器和逾時設定。

  • 如果[互助／平行列印]索引標籤上的所有設定都無法使用，請進行下列步驟。

    1. 按一下[取消]，關閉[連接埠設定：]對話方塊。

    2. 啟動SmartDeviceMonitor for Client，然後在工作列的SmartDeviceMonitor for Client圖示按一下滑鼠右鍵。

    3. 指向[內容]，再按[擴充功能設定]。

    4. 選擇[為每個連接埠設定互助／平行列印]核取方塊。

    5. 按一下[確定]，關閉[擴充功能設定]對話方塊。

  • 關於詳細資訊，請參閱「使用SmartDeviceMonitor for Client連接埠」或SmartDeviceMonitor for Client說明。
