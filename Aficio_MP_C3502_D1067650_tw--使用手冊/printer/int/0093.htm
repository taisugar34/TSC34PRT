傳送PDF檔

利用下列步驟，使用列印指令將PDF檔傳送至本機。

以下範例使用「lpr」指令指定機器的IP位址，以列印PDF檔案。

關於列印指令的詳細資訊，請參閱「UNIX supplement」。

%lpr -S "printer's IP address" [-Poption] [-o l]\ "path"\"PDF file name"

  • 以適當的設定值，取代引號中包括的字串。

  • 輸入方括號[ ]中包括的文字，是選擇性動作。
