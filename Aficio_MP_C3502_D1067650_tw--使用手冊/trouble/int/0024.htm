使用傳真機功能時顯示的訊息

本節介紹機器的主要訊息。如果顯示出其他訊息，請根據顯示的指示進行操作。

訊息

原因

解決方法

“驗證已失敗。”

輸入的登入使用者名稱或登入密碼不正確。

請聯絡使用者管理員。

“驗證已失敗。”

本機無法執行驗證。

請聯絡您的管理員。

“無法偵測原稿尺寸。選擇掃描尺寸。”

機器無法偵測原稿的尺寸。

在控制面板上的[掃描尺寸]之下，指定[掃描設定]中的掃描區域，然後重新傳送文件。

“找不到指定的路徑。請檢查設定。”

輸入做為目的地的電腦或資料夾名稱錯誤。

檢查目的地電腦名稱與資料夾名稱是否正確。

“檢查是否發生任何網路問題。”

[13-10]

您輸入的別名電話號碼已經在Gatekeeper上被其他裝置註冊。

  • 在[傳真功能]的[初始設定]中，選擇[H.323 設定]，然後查看是否正確設定了別名電話號碼。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[13-11]

無法存取Gatekeeper。

  • 在[傳真功能]的[初始設定]中，選擇[H.323 設定]，然後查看是否正確設定了Gatekeeper位址。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[13-17]

SIP伺服器拒絕了使用者名稱的登記。

  • 在[傳真功能]的[初始設定]中，選擇 [SIP 設定]，然後檢查「SIP 伺服器 IP 位址」和「SIP 使用者名稱」的設定是否正確。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[13-18]

無法存取SIP伺服器。

  • 在[傳真功能]的[初始設定]中，選擇[SIP 設定]，然後查看是否正確設定了「SIP 伺服器 IP 位址」。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[13-24]

為SIP伺服器登記的密碼和為本機登記的密碼不同。

請聯絡您的管理員。

“檢查是否發生任何網路問題。”

[13-25]

在[有效協定]中，IP位址並未啟用，或是登記了不正確的IP位址。

  • 在[系統設定]的[介面設定]中，選擇[有效協定]，然後查看是否將「IPv4」設定為「有效」。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 在[系統設定]的[介面設定]中，選擇[機器IPv4位址]，然後檢查IP位址的設定是否正確。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡您的管理員。

“檢查是否發生任何網路問題。”

[13-26]

「有效協定」和「SIP伺服器IP位址」的設定值不同，或是登記了不正確的IP位址。

  • 在[系統設定]的[介面設定]中，檢查IP位址的設定是否正確。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡您的管理員。

“檢查是否發生任何網路問題。”

[14-01]

找不到指定用於傳送的DNS伺服器、SMTP伺服器或資料夾，或是找不到繞過（非穿越）SMTP伺服器之網路傳真目的地。

  • 在[系統設定]的[介面設定]中，查看是否正確設定了DNS伺服器。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 在[系統設定]的[檔案傳送]中，選擇[SMTP伺服器名稱]，然後查看是否正確設定了「伺服器名稱」或「埠編號」。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 檢查是否正確指定了傳送用資料夾。

  • 檢查指定用於傳送的資料夾所在電腦是否正常運行。

  • 檢查LAN電纜是否正確連接至本機。

  • 聯絡目的地的網路管理員。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[14-09]

電子郵件傳送遭到SMTP驗證、SMTP之前的POP驗證，或是指定用於傳送的資料夾所在電腦的登入驗證拒絕。

  • 在[系統設定]的[檔案傳送]下，選擇 [SMTP驗證]、[SMTP之前的POP]或[傳真電子郵件帳號]，然後檢查「使用者名稱」和「密碼」的設定是否正確。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 檢查指定了自動轉傳用資料夾的電腦的使用者識別碼和密碼是否正確設定。

  • 檢查是否正確指定了自動轉傳用資料夾。

  • 檢查指定了自動轉傳用資料夾的電腦是否正常運行。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[14-33]

本機和管理員的電子郵件地址都沒有登記。

  • 在[系統設定]的[檔案傳送]下，選擇[傳真電子郵件帳號]，然後檢查「電子郵件地址」的設定是否正確。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[15-01]

未登錄POP3／IMAP4伺服器位址。

  • 在[系統設定]的[檔案傳送]中，選擇[POP3／IMAP4設定]，然後查看是否正確登錄了「伺服器名稱」。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[15-02]

無法登入POP3／IMAP4伺服器。

  • 在[系統設定]的[檔案傳送]下，選擇[傳真電子郵件帳號]，然後檢查「使用者名稱」和「密碼」是否正確登錄。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[15-03]

未登錄機器的電子郵件地址。

  • 在[系統設定]的[檔案傳送]下，檢查本機的電子郵件地址是否正確登錄。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[15-11]

找不到DNS伺服器或POP3／IMAP4伺服器。

  • 在[系統設定]的[介面設定]中，查看是否正確登錄了DNS伺服器的IP位址。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 在[系統設定]的[檔案傳送]中，選擇[POP3／IMAP4設定]，然後查看是否正確登錄了「伺服器名稱」。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 在[系統設定]的[檔案傳送]中，選擇[電子郵件通訊埠]，然後查看是否正確登錄了POP3／IMAP4伺服器的埠號。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 檢查LAN電纜是否正確連接至本機。

  • 在[系統設定]的[檔案傳送]中，選擇[接收協定]，然後查看是否正確登錄了接收協定。您也可以使用Web Image Monitor。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“檢查是否發生任何網路問題。”

[15-12]

無法登入POP3／IMAP4伺服器。

  • 在[系統設定]的[檔案傳送]下，選擇[傳真電子郵件帳號]，然後檢查「使用者名稱」和「密碼」是否正確登錄。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 在[系統設定]的[檔案傳送]下，檢查SMTP之前的POP驗證的使用者名稱和密碼是否正確登錄。您也可以使用Web Image Monitor進行確認。請參閱Web Image Monitor的說明。

  • 請聯絡網路管理員。

“連接到LDAP伺服器失敗。檢查伺服器狀態。”

出現網路錯誤，連線失敗。

  • 重試一次該操作。若仍顯示此訊息，則網路可能忙錄。

  • 在[系統設定]的[管理員工具]下，檢查LDAP伺服器的設定。關於詳細資訊，請參閱網路及系統設定參考手冊中的「設定LDAP伺服器」。

“發生錯誤，已取消傳送。”

  • 立即傳送過程中原稿夾紙。

  • 機器內部發生問題，或是電話線路上出現雜訊。

按[結束]，然後再次傳送文件。

“超出顯示搜尋結果的最大數目。最大值：n”

（n表示一個數字。）

搜尋結果數已經超出了可顯示的最大項目數。

更改搜尋條件後再次搜尋。

“超過LDAP伺服器搜尋時間限制。檢查伺服器狀態。”

出現網路錯誤，連線失敗。

  • 重試一次該操作。若仍顯示此訊息，則網路可能忙錄。

  • 在[系統設定]的[管理員工具]下，檢查LDAP伺服器的設定。關於詳細資訊，請參閱網路及系統設定參考手冊中的「設定LDAP伺服器」。

“發生功能上的問題。已停止處理。”

當機器正在接收網路傳真的文件時，關閉了主電源開關。

即使立即再開啟主電源開關，視郵件伺服器而定，若尚未超過逾時時間，機器可能無法繼續接收網路傳真。等到郵件伺服器的逾時時間超過，然後繼續接收網路傳真。關於詳細資訊，請聯絡您的網路管理員。

“傳真功能上的問題。資料將被初始化。”

傳真有問題。

記下顯示畫面上顯示的代碼編號，然後聯絡維修代表。仍可使用其他功能。

“LDAP伺服器驗證失敗。檢查設定。”

出現網路錯誤，連線失敗。

對LDAP伺服器驗證的使用者名稱和密碼進行正確設定。

“另一項功能正在掃描原稿。請切換到以下功能，然後按停止鍵取消掃描或按開始鍵繼續。”

目前正在使用傳真機功能以外的某項機器功能，例如，文件伺服器功能。

在以傳真方式傳送檔案之前，請先取消進行中的工作。例如，按下[結束]，然後按下[文件伺服器]鍵。接下來，按下[清除／停止]鍵。在出現“按了清除／停止鍵。您確定要停止掃描嗎？”訊息時，按下[停止]。

“放回原稿，檢查並按開始鍵。”

記憶傳送過程中原稿夾紙。

按[結束]，然後再次傳送文件。

“移除內部接紙盤1的紙張。”

內部接紙盤1已滿。

移除紙張。如果其他接紙盤也滿了，螢幕上顯示的接紙盤名稱會隨之改變。請移除螢幕上指示之接紙盤中的紙張。

“含有無效目的地。您要僅選擇有效目的地嗎？”

指定的群組包含傳真目的地、電子郵件目的地和／或資料夾目的地，其中一個目的地與指定的傳送方法不相容。

在每個傳送顯示的訊息中，按下[選擇]。

“部份頁面幾乎完全空白。如果要取消，請按停止鍵。”

文件的第一頁幾乎空白。

可能掃描到原稿的空白面。請務必正確放置原稿。關於判定空白紙頁原因的詳細資訊，請參閱傳真參考手冊中的「偵測空白頁」。

“部分頁面幾乎完全空白。”

文件的第一頁幾乎空白。

可能掃描到原稿的空白面。請務必正確放置原稿。關於判定空白紙頁原因的詳細資訊，請參閱傳真參考手冊中的「偵測空白頁」。

“選擇的檔案中包含沒有存取權限的檔案。僅刪除有存取權限的檔案。”

您嘗試刪除您沒有刪除權限的文件。

只有管理員能夠刪除此文件。如果您想要刪除沒有刪除權限的文件，請聯絡管理員。

“正在更新目的地清單。請稍候。指定的目的地或傳送者姓名被清除。”

目前正在使用Web Image Monitor，從網路更新目的地清單。

請等待直至此訊息消失。在顯示此訊息時，請勿關閉電源。視所需更新的目的地數目而定，在您繼續操作之前可能會有些延遲。當顯示此訊息時，無法進行操作。

“您沒有使用此功能的權限。”

登入的使用者名稱不具有所選功能的權限。

關於所需功能之權限的詳細資訊，請聯絡您的管理員。

  • 如果出現“檢查是否發生任何網路問題。” 訊息，表示機器並未正確連線至網路，或是機器的設定不正確。如果您不需要連接至網路，您可以指定設定，以不顯示此訊息，而且[傳真機]鍵不再亮起。關於如何執行此作業的詳細資訊，請參閱傳真參考手冊中的「參數設定」。若您將機器重新連線至網路，請務必配置適當的使用者參數以設定「顯示」。

  • 如果紙匣中的紙張用盡，顯示螢幕上就會出現“沒紙了。請放入紙張。”訊息，要求您添加紙張。如果其他紙匣中還剩餘紙張，即使顯示畫面上出現此訊息，您仍可和往常一樣接收文件。您可以使用「參數設定」開啟或關閉這項功能。關於如何執行此作業的詳細資訊，請參閱傳真參考手冊中的「參數設定」。
