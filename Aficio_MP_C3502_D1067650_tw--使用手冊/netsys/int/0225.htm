其他情況下

將網路印表機的IP位址和主機名稱新增到用於列印的電腦上的主機檔案中。視作業系統而定，新增的方法會不同。

例如，使用記事本檔案打開主機檔案。

若電腦的作業系統為Windows 2000，則主機檔案可在下列資料夾中找到：

\WINNT\SYSTEM32\DRIVERS\ETC\HOSTS

若電腦的作業系統為Windows XP／Vista／7、Windows Server 2003／2003 R2／2008／2008 R2，則主機檔案可在下列資料夾中找到：

\WINDOWS\SYSTEM32\DRIVERS\ETC\HOSTS

使用下列格式將 IPv4 或 IPv6 位址及主機名稱新增至主機檔案：

192.168.15.16 host # NP

「192.168.15.16」為IPv4位址，「host」為印表機的主機名稱，而「#NP」由註解替代。分別在「192.168.15.16」和「host」之間及「host」和「#NP」之間插入一個空格或定位點，對此格式使用單行。

2001:DB::100 host # NP

「2001:DB::100」為IPv6位址，「host」為印表機的主機名稱，而「#NP」由註解替代。分別在「2001:DB::100」和「host」之間及「host」和「#NP」之間插入一個空格或定位點，對此格式使用單行。

儲存檔案。

  • 在IPv6協定的Windows Server 2003／2003 R2／2008／2008 R2下使用主機名稱時，請使用外部DNS伺服器執行主機名稱解析。無法使用主機檔案。
