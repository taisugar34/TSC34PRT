使用Web Image Monitor

您可以使用Web Image Monitor，查看機器的狀態及變更設定。

可用的操作

使用Web Image Monitor可以從用戶端電腦遠端執行下列操作。

  • 顯示機器的狀態或設定

  • 查看列印工作狀態或記錄

  • 查看、修改、列印或刪除儲存在文件伺服器中的列印工作

  • 中止目前的列印工作

  • 重設本機

  • 管理通訊錄

  • 進行機器設定

  • 進行網路協定設定

  • 進行安全設定

設定本機

若要從Web Image Monitor執行操作，則需要TCP/IP。本機設定為使用TCP/IP後，即可從Web Image Monitor進行操作。

建議的網頁瀏覽器

  • Windows：

    Internet Explorer 5.5 SP2以上版本

    Firefox 1.0或更新版本

  • Mac OS：

    Firefox 1.0或更新版本

    Safari 1.0、1.2、2.0（412.2）或更新版本

Web Image Monitor支援螢幕閱讀器軟體。建議使用JAWS 7.0或更新的版本。

  • Safari無法在Mac OS X 10.4.1上使用。

  • 如果不啟用JavaScript和cookie，或使用非建議的網頁瀏覽器，則可能發生顯示和操作問題。

  • 如果使用代理伺服器，請變更網頁瀏覽器設定。關於設定的詳細資訊，請洽詢您的管理員。

  • 如果您按一下瀏覽器的返回按鈕，卻未出現上一頁，請按一下瀏覽器的重新整理按鈕並重試。

  • 機器資訊不會自動更新。若要執行更新，請點一下顯示區域中的[重新整理]。

  • 建議在同一個網路中使用Web Image Monitor。

  • 若機器受防火牆保護，便無法從防火牆外部的電腦存取。

  • 在DHCP下使用本機時，DHCP伺服器設定可能自動變更IP位址。在本機上啟用DDNS設定，然後使用本機的主機名稱進行連線。您也可在DHCP伺服器上設定一個靜態的IP位址。

  • 如果HTTP連接埠被停用，則無法使用本機的URL來建立連線。必須在本機上啟用SSL設定。關於詳細資訊，請諮詢您的網路管理員。

  • 使用SSL加密協定時，輸入「https://（機器的IP位址或主機名稱）/」。

  • 必須在電腦上安裝Internet Explorer。請使用最新版本。建議使用Internet Explorer 6.0或更新的版本。

  • 使用Firefox時，字體和顏色可能會有所不同，表格亦可能會變形。

  • 在IPv6協定的Windows Server 2003／2003 R2／2008／2008 R2下使用主機名稱時，請使用外部DNS伺服器進行主機名稱解析。無法使用主機檔案。

  • 若要在Web Image Monitor中使用JAWS 7.0，必須使用Windows作業系統和Microsoft Internet Explorer 5.5 SP2或更新的版本。

  • 如果您使用Internet Explorer 8，下載速度會比其他瀏覽器慢。若要加快Internet Explorer 8的下載速度，請開啟瀏覽器的[網際網路選項]選單，將機器的URL登記為信任的網站。接著停用信任的網站的SmartScreen篩選器。關於這些設定的詳細資訊，請參閱Internet Explorer 8說明檔案。

  • 將機器的URL登錄為書籤，可加快Web Image Monitor的存取速度。請注意，登記的URL必須為登記前所顯示頁面上方的URL。若您登記登入後所顯示頁面的URL，則無法從書籤正確開啟Web Image Monitor。
