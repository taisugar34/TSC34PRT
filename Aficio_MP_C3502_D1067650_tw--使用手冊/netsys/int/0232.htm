限制

下列限制適用Windows Terminal Service環境。

這些是Windows Terminal Service或Citrix Presentation Server／Citrix XenApp內建的限制。

Windows Terminal Service

在Windows Terminal Service環境中，有些印表機驅動程式的功能無法使用。

在安裝Windows Terminal Service的環境中，可使用部分印表機驅動程式的功能，即使不使用Windows Terminal Service的任何功能亦然。將SmartDeviceMonitor for Client安裝到Windows 2000 Server系列電腦上執行Terminal Service的環境中時，請確實使用安裝模式進行安裝。下列為使用安裝模式進行安裝的兩種方法：

  1. 使用[控制面板]中的[新增／刪除程式]來安裝SmartDeviceMonitor for Client。

  2. 在MS-DOS命令提示行輸入下列命令：

    CHANGE USER/INSTALL

若要離開安裝模式，在MS-DOS命令提示行輸入下列命令。

CHANGE USER/EXECUTE

Citrix Presentation Server／Citrix XenApp[自動建立用戶端印表機]

使用[自動建立用戶端印表機]，您可以選擇透過複製用戶端的本機印表機資料到MetaFrame伺服器建立的邏輯印表機。強烈建議您將此功能應用於工作之前，先在您的網路環境中進行測試。

  • 選購設備的設定在該設備連線中斷之後，將不會儲存在伺服器中。每次用戶端電腦登入到伺服器時，選購設備的設定都將恢復至其預設設定。

  • 當列印大量的點陣圖影像或者在WAN環境中透過ISDN等撥號線路使用伺服器時，視資料傳輸速率而定，列印可能會無法進行，或者可能會發生錯誤。

Citrix Presentation Server／Citrix XenApp[印表機驅動程式複製]

使用[印表機驅動程式複製]，您可以對伺服器範圍的所有伺服器散佈印表機驅動程式。強烈建議您將此功能應用於工作之前，先在您的網路環境中進行測試。

  • 如果未正確複製印表機驅動程式，請將這些驅動程式直接安裝到各個伺服器上。
