set

使用「set」命令，將協定資訊顯示設定為「up」或「down」。

檢視設定

下列命令顯示協定資訊（up/down）。

msh> set ipv4

msh> set ipv6

msh> set ipsec

msh> set netware

  • 只有安裝NetWare時，才可使用此命令。

msh> set smb

msh> set protocol

  • 如果指定協定，即出現TCP/IP、NetWare及SMB的相關資訊。

msh> set lpr6

設定

  • 輸入「up」啟用協定，而輸入「down」停用協定。

您可將協定設定為「有效」或「無效」。

msh> set ipv4 {up|down}

  • 如果停用IPv4，則登出後無法使用遠端存取。如果誤設了該項目，可以使用控制面板來啟用經由IPv4的遠端存取。

  • 停用IPv4同時也會停用lpr、ftp、rsh、diprint、web、snmp、ssl、ipp、http、bonjour、wsdev和sftp。

msh> set ipv6 {up|down}

  • 如果停用IPv6，則登出後無法使用遠端存取。如果誤設了該項目，可以使用控制面板來啟用經由IPv6的遠端存取。

  • 停用IPv6同時也會停用lpr6、ftp6、rsh6、diprint6、ssl6、ipp6、http6、bonjour6、wsdev6和sftp6。

msh> set ipsec {up|down}

msh> set netware {up|down}

  • 只有安裝NetWare時，才可使用此命令。

msh> set smb {up|down}

msh> set lpr {up|down}

msh> set lpr6 {up|down}

msh> set ftp {up|down}

msh> set ftp6 {up|down}

msh> set rsh {up|down}

msh> set rsh6 {up|down}

msh> set diprint {up|down}

msh> set diprint6 {up|down}

msh> set web {up|down}

msh> set snmp {up|down}

msh> set ssl {up|down}

msh> set ssl6 {up|down}

  • 如果印表機不可使用Secured Sockets Layer（SSL，一種加密協定）功能，則無法啟用它來使用該功能。

msh> set nrs {up|down}

msh> set ipp {up|down}

msh> set ipp6 {up|down}

msh> set http {up|down}

msh> set http6 {up|down}

msh> set bonjour {up|down}

msh> set bonjour6 {up|down}

msh> set ssh {up|down}

msh> set ssdp {up|down}

msh> set nbt {up|down}

msh> set sftp {up|down}

msh> set sftp6 {up|down}

msh> set wsdev {up|down}

msh> set wsdev6 {up|down}

  • 如果同時啟用「wsdev」和「wsdev6」，則這兩項會同時在協定資訊顯示中顯示為「up」，但是同時對WSD（裝置）、WSD（印表機）和WSD（掃描器）使用IPv4。
