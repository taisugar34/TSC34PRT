SNMP

使用SNMP管理器可以獲得有關於本機的資訊。

在UDP和IPX上執行的SNMP代理器已結合到本機的內建乙太網路卡。

本機器也支援SNMPv3，增強使用者驗證、資料加密和存取控制安全性。

若要透過SNMPv3把通訊加密，必須指定本機器的加密密碼。

  • 如果變更本機的社群名稱，請使用SNMP設定工具相應變更連線的電腦的設定。關於詳細資訊，請參閱SNMP設定工具的說明。

  • 若要使用SNMP設定工具，您必須先從供應商網站下載SmartDeviceMonitor for Admin並安裝至電腦。如需下載SmartDeviceMonitor for Admin的相關資訊，請聯絡您的當地經銷商。

預設的社群名稱為[public]和[admin]。您可以使用這些社群名稱來獲得MIB資訊。

啟動SNMP設定工具

  • Windows 2000：

    按一下[開始]鍵。

    指向[程式]選單上的[SmartDeviceMonitor for Admin]。

    按一下[SNMP設定工具]。

  • Windows XP／Vista／7、Windows Server 2003／2003 R2／2008／2008 R2

    按一下[開始]鍵。

    指向[所有程式]選單上的[SmartDeviceMonitor for Admin]。

    按一下[SNMP設定工具]。
