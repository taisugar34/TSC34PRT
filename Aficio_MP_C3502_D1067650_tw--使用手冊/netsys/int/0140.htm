設定網路介面板

使用下列指令顯示網路介面板設定：

  • telnet：使用「show」指令。

Protocol Up／Down

即顯示下列協定（「Up」代表啟用；「Down」代表停用）：。

IPv4、IPv6、IPsec、NetWare、SMB

Device Up／Down

即顯示下列裝置（「Up」代表啟用；「Down」代表停用）：。

Parallel、USB

Ethernet interface

顯示乙太網路的速度。

Syslog priority

即顯示「LOG_INFO」。

NVRAM version

顯示NVRAM版本。

Device name

顯示裝置名稱。

Comment

顯示備註。

Location

顯示裝置位置。

Contact

顯示聯絡人詳細資訊。

Soft switch

顯示觸控開關。

TCP／IP

即顯示下列項目：（「Up」代表啟用；「Down」代表停用）：。

  • Mode (IPv4)

  • Mode (IPv6)

    ftp、lpr、rsh、diprint、web、http、ftpc、snmp、ipp、autonet、Bonjour、ssl、nrs、rfu、nbt、ssdp、ssh、sftp、WSD (Device)、WSD (Printer)、WSD (Scanner)、rhpp

  • IPv4

    DHCP、Address、Netmask、Broadcast、Gateway

  • IPv6

    Stateless、Manual、Gateway、DHCPv6-lite、DUID、DHCPv6 option lifetime

  • IPsec

    Manual Mode、Excluded Protocol （https、dns、dhcp、wins）

  • EncapType

  • Host name

  • DNS Domain

  • Access Control

    IPv4：Access Entry [1]－[5]

    IPv6：Access Entry [1]－[5]

  • SNTP Server

  • Time Zone

  • SNTP Server polling time

  • SYSLOG server

  • Home page URL1

  • Home page link name1

  • Home page URL2

  • Home page link name2

  • Help page URL

  • RHPP Port

  • RHPP timeout

NetWare

即顯示下列項目：

  • EncapType

  • RPRINTER number

  • Print server name

  • File server name

  • Context name

  • Switch

    Mode、NDS/Bindery、Packet negotiation、Login Mode

  • Print job timeout

  • Protocol

  • SAP interval time

  • NDS Tree Name

  • Transfer Protocol

SMB

即顯示下列項目：

  • Switch

    Mode、Direct print、Notification

  • Workgroup name

  • Computer name

  • Comment

  • Share name [1]

  • Protocol

Ethernet

即顯示下列項目：

802.1X Authentication

DNS

即顯示下列項目：

  • IPv4

    Server [1]-[3], Selected IPv4 DNS Server

  • IPv6

    Server [1]-[3]、Selected IPv6 DNS Server、Resolver Protocol

Domain name

即顯示下列項目：

ether

DDNS

即顯示下列項目：

ether

WINS

即顯示下列項目：

  • ether

    Primary WINS、Secondary WINS

SSDP

即顯示下列項目：

UUID、Profile、TTL

UPnP

即顯示下列項目：

URL

Bonjour

即顯示下列項目：

Computer Name (cname)、Local Hostname (ether)、Location、Priority (diprint)、Priority (lpr)、Priority (ipp)、IP TTL、LinkLocal Route for Multi I/F、IPP Port

SNMP

即顯示下列項目：

SNMPv1v2、SNMPv3、protocol、v1Trap、v2Trap、v3Trap、SNMPv1v2 Remote Setting、SNMPv3 Privacy

ssh

即顯示下列項目：

Compression、Port、TimeOut、LoginTimeOut

AuthFree

即顯示下列項目：

  • IPv4

    AuthFree Entry [1]－[5]

  • IPv6

    AuthFree Entry [1]－[5]

  • Parallel

  • USB

LPR

即顯示下列項目：

lprm check host、lpr prnerrchk

Certificate

即顯示下列項目：

Verification

WS-MFP

即顯示下列項目：

Network Device Name、Comments、Location、Presentation URL、WSD (Device) TCP Port、WSD (Printer) TCP Port、WSD (Printer) Job Timeout、WSD (Scanner) TCP Port、MetadataVersion、UUID

IEEE 802.1X

即顯示下列項目：

User name、Domain name、EAP Type、Password、Phase 2 user name、Phase 2 Method TTLS、Phase 2 Method PEAP、Server cert、Intermediate CA、Server ID、Sub domain

Shell mode

即顯示「使用者模式」。
