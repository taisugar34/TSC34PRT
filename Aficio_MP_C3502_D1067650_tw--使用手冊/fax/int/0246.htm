設定選購件的組態設定

完成本機器所安裝的選購件的設定。

在[開始]選單中，選擇[控制台]。然後選擇[印表機和其他硬體]，再選擇[印表機和傳真]。

如果是Windows 2000，請指向[開始]選單的[設定]，然後按一下[印表機]。

在Windows XP Professional或Windows Server 2003／2003 R2中，選擇[開始]選單上的[印表機和傳真]。

如果是Windows Vista或Windows Server 2008，請選擇[開始]選單的[控制台]。然後選擇[硬體和音效]，再選擇[印表機]。

在Windows 7，請選擇[開始]選單的[裝置和印表機]。

選擇使[LAN－Fax M8]反白顯示，然後按一下[檔案]選單中的[內容]。

在Windows 7，請在[LAN－Fax M8]上按一下滑鼠右鍵，然後按一下[印表機內容]。

按一下[配件]索引標籤，然後設定選購件的組態設定。

勾選已安裝選購件的核取方塊。

按一下[套用]。

選購件的組態設定完成。

按一下[確定]。

[配件]索引標籤

除了選購件組態設定項目之外，[配件]索引還包含下列項目。

  • 禁止手動輸入目的地

    勾選這個選項可禁止直接指定目的地。

  • 啟用電子郵件

    以LAN－Fax功能使用網路傳真時，請勾選這個選項。

  • IP－Fax

    使用IP－Fax時，請勾選這個選項。

    勾選這個選項後，請按一下適當的選項按鈕選擇通訊協定。

  • 如果未依指示設定本機的選購件，可能導致LAN－Fax功能失效。

  • 若本機已連接網路，且您的電腦上安裝了SmartDeviceMonitor for Client，將會自動對機器上所安裝的各選購件進行設定。如果設定值與所安裝的選購件不符，請按一下[從設備載入]。
