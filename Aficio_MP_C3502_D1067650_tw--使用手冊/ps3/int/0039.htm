安裝PostScript 3印表機驅動程式（Windows XP、Windows Server 2003 ／ 2003 R2）

本節說明將印表機用作Windows網路印表機使用時在Windows XP或Windows Server 2003 ／2003 R2中安裝PostScript 3印表機驅動程式的方法。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

  • 如果使用SmartDeviceMonitor for Client連接埠，透過機器的列印伺服器進行列印，則用戶端電腦無法使用恢復列印和平行列印。

  • 如果在Windows XP或Windows Server 2003 ／ 2003 R2的列印伺服器上進行列印，則用戶端電腦可能無法使用SmartDeviceMonitor的通知功能。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

請結束目前執行的所有應用程式。

將隨附的光碟片置入電腦的光碟機。

安裝程式隨即啟動。

選擇介面語言，然後按下[確定]。

將安裝所選語言版本的印表機驅動程式。

按一下[PostScript 3印表機驅動程式]。

新增印表機精靈啟動。

按一下[下一步>]。

按一下[網路印表機或接至其他電腦的印表機]，然後按一下[下一步>]。

選擇[瀏覽印表機]，然後按一下[下一步>]。

在[共用印表機:]視窗中，按兩下您要用作列印伺服器的電腦名稱。

選擇您要使用的印表機，然後按[下一步>]。

若出現安全警告訊息，請按一下[是]。

印表機驅動程式開始安裝。

按一下[下一步>]。

選擇[是]將本機設為預設值。

按一下[完成]。

重新啟動電腦，以便完成安裝。

  • 若要停止安裝選取的驅動程式，請在完成安裝之前，按一下[取消]。

  • 使用者代碼可以在印表機驅動程式安裝以後設定。關於使用者代碼的詳細資訊，請參閱印表機驅動程式說明。

  • 某些作業系統設定可能無法使用自動執行。此時，啟動光碟片根目錄中的「Setup.exe」。
