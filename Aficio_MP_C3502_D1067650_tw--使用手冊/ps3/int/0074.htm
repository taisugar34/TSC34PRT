使用Bonjour

依照下列程序在Mac OS X 10.2.3或以上版本中使用Bonjour進行列印。下列步驟以Mac OS X 10.3.7作為範例。您也可以使用乙太網路連線。

啟動系統偏好設定。

按一下[列印和傳真]。

按一下[設定印表機]。

根據電腦使用的作業系統，可能不會顯示[設定印表機]。若未顯示[設定印表機]，請繼續下列步驟。

按一下[新增]或[]按鈕。

Mac OS X 10.5或更新版本

按一下[預設]。若未顯示本機名稱，請依據您的網路環境（TCP／IP等）選擇相對應的圖示。

Mac OS X 10.4

按一下「連接」欄中顯示「Bonjour」的印表機。

其他Mac OS X

在彈出式選單上，按一下[Rendezvous]。

從[印表機型號：]清單中，選擇已連接印表機的名稱，然後按一下[新增]。

若使用Mac OS X 10.4版本，將出現可安裝選購件視窗。選擇要設定的選購件，為其選擇合適的設定，然後選擇[繼續]。

若使用Mac OS X 10.5或更新版本，請在[印表機名稱]清單中，選擇您所用的印表機，然後在[類型]中選擇Bonjour。

結束系統偏好設定。

  • 當以Macintosh電腦的Rendezvous連接進行列印時，印表機語言不會自動變更。請在列印前，用本機的控制面板將印表機語言變更為[自動偵測]或[PS]。
