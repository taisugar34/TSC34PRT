安裝PostScript 3印表機驅動程式（Windows Vista、Windows Server 2008）

本節說明如何在Windows Vista或Windows Server 2008中安裝使用WSD 連接埠的PostScript 3 印表機驅動程式。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

請結束目前執行的所有應用程式。

按一下[開始]，然後按一下[網路]。

出現[網路]視窗，自動開始搜尋裝置。

在印表機圖示上，按一下滑鼠右鍵，再按[安裝]。

如果出現[使用者帳號控制]對話方塊，請按[是]或[繼續]。

出現[發現新硬體]對話方塊。

按一下[找到並安裝驅動程式軟體（建議選項）]。

如果出現[使用者帳號控制]對話方塊，請按[是]或[繼續]。

按一下[瀏覽我的電腦搜尋驅動程式軟體（進階）]。

將隨附的光碟片置入電腦的光碟機。

如果出現[自動播放]對話方塊，請按一下[關閉]按鈕。

按一下[瀏覽‧‧‧]，指定印表機驅動程式的位置。

若光碟機代號為D，印表機驅動程式的來源檔案則位於「D:\DRIVERS\PS\(Language)\XP_VISTA\DISK1」。

按一下[下一步]。

如果出現[Windows無法驗證此驅動程式軟體的發行者]訊息，請按[仍安裝此驅動程式軟體]。

按一下[關閉]。

若安裝成功，以WSD連接埠連接的印表機，其圖示將新增至[印表機]視窗中。

  • 「WSD」後面的連接埠名稱採用隨機字元字串。無法任意更改。

  • 若要停止安裝，在安裝完成前，按一下[取消]。當重新安裝WSD連接埠時，在[網路]視窗中以滑鼠右鍵按一下本機圖示，然後按一下[解除安裝]。
