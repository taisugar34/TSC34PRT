安裝PostScript 3印表機驅動程式（Windows XP、Windows Server 2003 ／ 2003 R2）

本節說明如何在Windows XP 或Windows Server 2003／2003 R2 中安裝使用標準TCP／IP 連接埠的PostScript 3 印表機驅動程式。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

  • 在IPv6環境中，您無法使用標準TCP／IP連接埠。請使用SmartDeviceMonitor for Client連接埠。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

請結束目前執行的所有應用程式。

將隨附的光碟片置入電腦的光碟機。

安裝程式隨即啟動。

選擇介面語言，然後按下[確定]。

將安裝所選語言版本的印表機驅動程式。

按一下[PostScript 3印表機驅動程式]。

新增印表機精靈啟動。

按一下[下一步>]。

按一下[連接此電腦的本機印表機]，然後按一下[下一步>]。

按一下[建立新的連接埠：]。

在[連接埠類型:]中按一下[標準TCP／IP連接埠]，然後按一下[下一步>]。

在[新增標準TCP／IP印表機連接埠精靈]對話方塊中，按一下[下一步>]。

在[印表機名稱或IP位址]方塊中輸入本機名稱或IP位址。

[連接埠名稱]文字方塊將自動取得連接埠名稱。如有需要，請變更此名稱。

按一下[下一步>]。

在[新增標準TCP／IP印表機連接埠精靈]對話方塊中，請按[完成]。

確認選擇了您要安裝驅動程式的機器名稱，然後按一下[下一步 >]。

如果需要，請變更機器名稱，然後按[下一步>]。

選擇[是]將本機設為預設值。

指定是否共用機器，然後按[下一步>]。

指定是否列印測試頁，然後按一下[下一步 >]。

按一下[完成]。

如果出現[硬體安裝]訊息，按一下[繼續安裝]。

印表機驅動程式開始安裝。

  • 若要停止安裝選取的驅動程式，請在完成安裝之前，按一下[取消]。

  • 使用者代碼可以在印表機驅動程式安裝以後設定。關於使用者代碼的詳細資訊，請參閱印表機驅動程式說明。

  • 某些作業系統設定可能無法使用自動執行。此時，啟動光碟片根目錄中的「Setup.exe」。
