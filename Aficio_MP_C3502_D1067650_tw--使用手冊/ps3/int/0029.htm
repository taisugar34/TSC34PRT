安裝PostScript 3印表機驅動程式（Windows Vista／7、Windows Server 2008 － TCP／IP）

本節說明如何在Windows Vista／7或Windows Server 2008中安裝使用SmartDeviceMonitor for Client連接埠的PostScript 3印表機驅動程式。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

請結束目前執行的所有應用程式。

將隨附的光碟片置入電腦的光碟機。

安裝程式隨即啟動。

選擇介面語言，然後按下[確定]。

將安裝所選語言版本的印表機驅動程式。

按一下[PostScript 3印表機驅動程式]。

新增印表機精靈啟動。

按一下[新增本機印表機]。

按一下[建立新的連接埠:]，然後選擇[連接埠類型]中的[SmartDeviceMonitor]。

按一下[下一步]。

按一下[TCP／IP]，然後按一下[搜尋]。

即會列出使用TCP／IP的機器。

選擇要使用的機器，然後按一下[確定]。

只有回應電腦廣播指令的印表機才會出現。若要使用未在此處列出的設備，按一下[指定位址]，然後輸入入機器的IP 位址或主機名稱。

確認選擇了您要安裝驅動程式的機器名稱，然後按一下[下一步]。

如有需要，請變更機器名稱，然後按[下一步]。

若出現將本機設為預設印表機的核取方塊，請視需要進行設定。

如果出現[Windows無法驗證此驅動程式軟體的發行者]訊息，請按[仍安裝此驅動程式軟體]。

指定是否共用機器，然後按[下一步]。

指定是否列印測試頁，然後按一下[完成]。

若出現將本機設為預設印表機的核取方塊，請視需要進行設定。

印表機驅動程式開始安裝。

  • 若要停止安裝選取的驅動程式，請在完成安裝之前，按一下[取消]。

  • 依據電腦作業系統的不同，可能會顯示[自動播放]對話方塊。出現時，請按一下[執行SETUP.EXE]。若出現[使用者帳戶控制]對話方塊，請按一下[是]或[繼續]允許執行自動播放程式。

  • 使用者代碼可以在印表機驅動程式安裝以後設定。關於使用者代碼的詳細資訊，請參閱印表機驅動程式說明。

  • 某些作業系統設定可能無法使用自動執行。此時，啟動光碟片根目錄中的「Setup.exe」。
