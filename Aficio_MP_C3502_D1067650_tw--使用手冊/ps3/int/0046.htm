Windows Vista、Windows Server 2008 － USB

本節說明如何在Windows Vista或Windows Server 2008中安裝PostScript 3印表機驅動程式以使用USB介面。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

若已安裝印表機驅動程式，將啟用隨插即用功能，並且連接到「USB」連接埠的機器的圖示會被新增至[印表機]視窗中。

印表機的驅動程式可以從本機提供的光碟片安裝。

如果未安裝印表機驅動程式，請依照本機的隨插即用指示從隨附的光碟片安裝。

透過USB電纜連接本機和電腦。

確實連接USB電纜。

在[尋找新增硬體精靈]視窗中，選擇[尋找並安裝驅動程式軟體（建議選項）]。

如果出現[使用者帳號控制]視窗，請按一下[是]或[繼續]。

當出現提示置入光碟片的訊息時，將光碟片放入光碟機。

如果出現[自動播放]對話方塊，請按一下[關閉]按鈕。

選擇您要安裝驅動程式的機器名稱，然後按一下[下一步]。

印表機驅動程式將自動開始安裝。

如果安裝時，出現[Windows無法驗證此軟體的發行者]畫面，請選擇[繼續安裝此驅動程式軟體]。

按一下[關閉]。

若安裝成功，以「USB001」連接埠連接的機器，其圖示將新增至[印表機]視窗中。

  • 若在網路中找到符合本機的印表機驅動程式，則不會顯示放入光碟的訊息。

  • 「USB」後方的數字將依據連接的印表機數量而變。
