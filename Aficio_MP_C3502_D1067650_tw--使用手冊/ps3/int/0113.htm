色彩設定檔

使用此項選擇色彩設定檔的樣式。

下表顯示了您可以選擇此功能的位置。

Windows

在[Print Quality]選單上，按一下[細節...]，可開啟一個對話方塊，然後在[色彩設定檔:]清單中，選擇項目。

Mac OS X

[色彩設定檔]會顯示在列印對話方塊中[印表機功能]內[功能設定：]清單的[一般2]中。

您可以選擇以下項目：

自動

使用此設定根據列印文件的外觀自動配置最佳的色彩設定檔樣式。

照片

使用此設定增強包含中間調的圖形和照片的再現。

簡報

使用此設定增強包含文字和圖形的文件的再現。此CRD最適合列印彩色圖表、簡報素材等等。如果您使用此CRD列印照片，色彩或漸層可能無法正常再現。

Solid Color

使用此設定列印特定的顏色、標誌等等。

使用者設定

使用此設定以從應用程式中下載的CRD列印影像。

CLP模擬1

使用此設定更加明亮和生動地列印藍色。

CLP模擬2

使用此設定更加明亮和生動地列印藍色。列印的顏色淺於CLP模擬1。

啟動ICM處理

此設定可啟動影像色彩管理。此設定僅適用於Windows作業系統。

  • 使用此項選擇演色字典（CRD）。CRD是指配色，因此請為列印的文件選擇正確的CRD。將[色彩設定：]設為[精細]或[超細]時，也會參照所選的CRD。您可使用下列項目：[自動]、[照片]、[簡報]和[Solid Color]。
