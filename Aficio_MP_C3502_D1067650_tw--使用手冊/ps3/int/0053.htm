如果在安裝時出現訊息

訊息編號58或34表示印表機驅動程式無法以自動執行安裝。請以[新增印表機]或[安裝印表機]來安裝印表機驅動程式。

對於Windows 2000：

在[開始]選單中，指向[設定]，再按[印表機]。

按兩下新增印表機的圖示。

按照新增印表機精靈中的指示進行操作。

若印表機驅動程式位於光碟中，PostScript 3印表機驅動程式的位置在D:\DRIVERS\PS\XP_VISTA\(Language)\DISK1。

對於Windows XP Professional和Windows Server 2003 ／ 2003 R2：

在[開始]選單上，按一下[印表機和傳真]。

按一下[新增印表機]。

按照新增印表機精靈中的指示進行操作。

若印表機驅動程式位於光碟中，PostScript 3印表機驅動程式的位置在D:\DRIVERS\PS\XP_VISTA\(Language)\DISK1。

對於Windows XP Home Edition：

在[開始]選單上，按一下[控制台]。

按一下[印表機和其他硬體]。

按一下[印表機和傳真]。

按一下[安裝印表機]。

按照新增印表機精靈中的指示進行操作。

若印表機驅動程式位於光碟中，PostScript 3印表機驅動程式的位置在D:\DRIVERS\PS\XP_VISTA\(Language)\DISK1。

對於Windows Vista與Windows Server 2008：

在[開始]選單上，按一下[控制台]。

出現[控制台]視窗。

按一下「硬體和音效」中的[印表機]。

按一下[新增印表機]。

按照新增印表機精靈中的指示進行操作。

若印表機驅動程式位於光碟中，PostScript 3印表機驅動程式的位置在D:\DRIVERS\PS\XP_VISTA\(Language)\DISK1。

對於Windows 7：

在[開始]選單上，按一下[控制台]。

按一下[硬體和音效]。

按一下[新增印表機]。

按照新增印表機精靈中的指示進行操作。

若印表機驅動程式位於光碟中，PostScript 3印表機驅動程式的位置在D:\DRIVERS\PS\XP_VISTA\(Language)\DISK1。

  • 如果安裝程式啟動，按一下[取消]退出。
