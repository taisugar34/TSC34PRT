使用IPP連接埠

本節說明如何安裝使用IPP 連接埠的PostScript 3 印表機驅動程式。

  • 安裝驅動程式需具備管理印表機的權限。請以系統管理員群組成員的身份登入。

  • 您可以從本機隨附的光碟片安裝印表機驅動程式，或從製造商網站下載驅動程式。

  • 如果使用的作業系統為64位元版本，請至製造商的網站下載印表機驅動程式。選擇本機及您使用的作業系統，然後下載驅動程式。

  • 在IPv6環境中，您無法使用IPP連接埠。請使用SmartDeviceMonitor for Client連接埠。

  • 您無法在Windows 2000中使用IPP連接埠。若是使用Windows 2000，請使用SmartDeviceMonitor for Client連接埠安裝PostScript 3印表機驅動程式。若要使用SmartDeviceMonitor for Client連接埠，您必須先從製造商的網站下載SmartDeviceMonitor for Client。關於SmartDeviceMonitor for Client連接埠的詳細資訊，請參閱「使用SmartDeviceMonitor for Client連接埠」。

  • 若要透過IPP－SSL進行列印，請使用SmartDeviceMonitor for Client連接埠。

  • 若要在Windows Vista／7或Windows Server 2008／2008 R2中透過IPP－SSL進行列印，您必須先在電腦上安裝憑證，才能使用IPP連接埠安裝印表機驅動程式。關於詳細資訊，請洽詢您的管理員。

  • 使用SmartDeviceMonitor for Client連接埠
