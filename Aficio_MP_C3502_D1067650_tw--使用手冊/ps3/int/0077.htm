工作種類

使用此項選擇列印工作的種類。

下表顯示了您可以選擇此功能的位置。

Windows

[工作種類：]會顯示在[列印喜好設定]對話方塊的[單鍵預設]索引標籤中。

Mac OS X

[工作種類：]顯示在列印對話方塊的[工作紀錄檔]中。

您可以選擇以下項目：

一般列印

選擇此項進行一般列印。列印工作在收到列印指令後立即開始。

樣本列印

使用本功能對多個列印工作僅列印一份。

其他份數將儲存到本機。儲存的工作可以從本機的控制面板進行列印。您也可以刪除儲存的工作。

「使用者識別碼」最多可以包含８個文數字（a－z、A－Z、0－9）字元。

輸入「使用者識別碼」可以幫您將您的列印工作與其他人的工作進行區分。

關於樣本列印功能的詳細資訊，請參閱「如何使用樣本列印功能」。

機密列印

使用本功能將文件以密碼儲存到本機記憶體中，然後根據需要進行編輯和列印。

「使用者識別碼」最多可以包含８個文數字（a－z、A－Z、0－9）字元。「密碼：」必須是4－8位數。

輸入「使用者識別碼」可以幫您將您的列印工作與其他人的工作進行區分。

關於機密列印功能的詳細資訊，請參閱「如何使用機密列印功能」。

保留列印

使用此功能可臨時將檔案儲存在本機中，稍後再從電腦或本機的控制面板進行列印。

「使用者識別碼」最多可以包含８個文數字（a－z、A－Z、0－9）字元。

「檔案名稱：」最多可以包含16個文數字（a－z、A－Z、0－9）字元。

輸入「使用者識別碼」可以幫您將您的列印工作與其他人的工作進行區分。

關於保留列印功能的詳細資訊，請參閱「如何使用保留列印功能」。

儲存列印

使用此功能可將檔案儲存在本機中，然後再從電腦或本機的控制面板進行列印。

「使用者識別碼」最多可以包含８個文數字（a－z、A－Z、0－9）字元。「密碼：」必須是4－8位數。

檔案名稱：最多可以包含16個文數字（a- z、A- Z、0-9）字元。

您可以為儲存的文件指定密碼，不過也可以不指定密碼。

輸入「使用者識別碼」可以幫您將您的列印工作與其他人的工作進行區分。

有關儲存列印功能的詳細資訊，請參閱「如何使用儲存列印／儲存並列印功能」。

儲存並列印

使用本功能立即列印檔案，同時將檔案儲存到印表機。

「使用者識別碼」最多可以包含８個文數字（a－z、A－Z、0－9）字元。「密碼：」必須是4－8位數。

「檔案名稱」最多可以包含16個英數（a- z、A- Z、0-9）字元。

您可以為儲存的文件指定密碼，不過也可以不指定密碼。

輸入「使用者識別碼」可以幫您將您的列印工作與其他人的工作進行區分。

有關儲存並列印功能的詳細資訊，請參閱「如何使用儲存列印／儲存並列印功能」。

文件伺服器

使用此功能將您想列印的文件儲存在印表機中，亦可儲存您之後想合併或處理列印的文件。

「檔案名稱：」最多可以包含16個文數字（a－z、A－Z、0－9）字元。

有關文件伺服器功能的詳細說明，請參閱印表機參考手冊的「存取文件伺服器」。
