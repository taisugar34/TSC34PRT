進行印表機的選項設定

當停用雙向通訊時，使用印表機驅動程式進行機器的選項設定。

當停用雙向通訊時，進行選項設定。

  • 變更印表機內容，需要使用管理印表機權限。請以系統管理員群組成員的身份登入。

在[開始]選單上，按一下[印表機和傳真]。

  • 使用Windows 2000時，在[開始]選單上，指向[設定]，然後按一下[印表機]。

  • 使用Windows Vista或Windows Server 2008時，在[開始]選單上，按一下[控制台]，然後按一下「硬體和音效」中的[印表機]。

  • 使用Windows 7或Windows Server 2008 R2時，按一下[開始]選單，然後按一下[裝置和印表機]。

按一下所要使用的印表機的圖示。

在[檔案]選單上，按一下[內容]。

  • 使用Windows Vista或Windows Server 2008時，在要使用的印表機圖示上，按一下滑鼠右鍵，再按[內容]。

  • 使用Windows 7或Windows Server 2008 R2時，在要使用的印表機圖示上，按一下滑鼠右鍵，再按[印表機內容]。

按一下[裝置設定]索引標籤。

從[可安裝選項]區域中選擇安裝的選項，然後進行必要的設定。

按一下[套用]，然後按[確定]，關閉印表機內容對話方塊。

  • 關於使用Mac系統指定機器的選項設定，請參閱「設定Mac OS X的選項」。
