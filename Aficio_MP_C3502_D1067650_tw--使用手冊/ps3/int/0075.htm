設定PostScript 3印表機驅動程式的選項

若要正確使用安裝的選購件，請先設定印表機驅動程式。如果選購件未被辨識，即使安裝了也無法使用。

  • 變更印表機內容，需要使用管理印表機權限。請以系統管理員群組成員的身份登入。

  • 若您使用Adobe PageMaker 6.0、6.5或7.0，則需在Adobe PageMaker的列印對話方塊內設定選購件。

下表說明各種作業系統的印表機驅動程式設定方法的位置。

設定印表機驅動程式的步驟因作業系統而不同。

此處以Windows XP及Mac OS X 10.5作為各作業系統系列的範例。

Windows作業系統

在印表機內容的[裝置設定]索引標籤中，您可在[可安裝的選項]中設定任何選項。

Mac OS X

在系統偏好設定的[列印和傳真]中，您可在[選項和耗材‧‧‧]中設定所有選項。

  • 若要設定選購件，請從Windows中存取印表機驅動程式。如果您從應用程式中存取印表機驅動程式，您將無法設定選購件。

  • 為瞭解您的列印環境適用的安裝方法，請參閱「進行印表機的選項設定」或「設定Mac OS X的選項」。
