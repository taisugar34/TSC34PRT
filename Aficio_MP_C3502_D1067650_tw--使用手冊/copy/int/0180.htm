列印儲存的文件

列印儲存的文件。

Web Image Monitor可讓您從電腦列印儲存在文件伺服器中的文件。

選擇要列印的文件。

如果已設定密碼，請輸入密碼，然後按下[確定]。

若要一次列印兩份或多份文件，請重複步驟1。

最多可列印30份文件。

指定列印條件時，請按下[至列印畫面]。

列印條件畫面隨即顯示。將會顯示文件的使用者名稱、檔案名稱和列印順序。

使用數字鍵輸入列印份數。

可設定的數量上限為999。

按下[開始]鍵。

開始列印。

  • 可使用畫面左邊的[使用者名稱]或[檔案名稱]搜尋目標文件。關於搜尋目標文件的詳細資訊，請參閱「搜尋儲存的文件」。

  • 您可以按照使用者名稱、檔案名稱，或建立日期將文件排序。在清單頂端，按下用來排序的項目。

  • 若要取消選擇，請按下已選擇的（反白）鍵。

  • 由於尺寸或解析度不同，某些選擇的文件可能無法列印。

  • 變更列印順序時，請取消反白顯示，然後指定正確的順序。

  • 按下[總清除]鍵取消各項選擇。

  • 按下[順序]，依列印順序顯示選擇的文件。

  • 操作結束後，影印和印表機功能會保留指定的列印條件，並於下次列印時套用。傳真功能沒有儲存列印條件的功能。

  • 指定兩份或多份文件時，列印條件會儲存在第一份文件，而非後續文件上。

  • 以下為可使用的列印條件設定。關於各項列印結果的詳細資訊，請參閱「基本影印」與「進階影印」。

    • 紙匣

    • 影本數目

    • 雙面影印（上到上、上到下）、騎馬釘裝訂、雜誌

    • 印後處理（分頁／交叉分頁／分類／裝訂／打孔）

    • 封面／隔頁紙（封面、封面／封底、隔頁／章節、隔頁紙）

    • 編輯／戳記（邊界調整、戳記）

  • 選擇分頁功能時若列印兩份或多份影本，您可僅列印一份以確認列印結果。關於樣本列印功能的詳細資訊，請參閱「樣本列印」。

  • 一次列印兩份或多份文件時，可指定列印順序，將多份文件當成一份連續的文件列印。

  • 一次列印兩份或多份文件時，針對第一份文件設定的列印條件，會套用到之後的所有文件上。

  • 選擇兩份或多份文件時，請按下[] []，確認文件的使用者名稱、文件名稱和列印順序。

  • 按下[檔案清單]還原文件選擇畫面。

  • 使用傳真機功能儲存的文件，無法指定戳記功能。

  • 您可以在使用者工具中的影印數量上限下變更影印數量上限。關於影印數量上限的詳細資訊，請參閱「一般功能」。

  • Web Image Monitor可讓您在電腦上列印儲存在文件伺服器中的文件。關於Web Image Monitor啟動程序的詳細資訊，請參閱「使用 Web Image Monitor顯示儲存文件」。
